My Project

Web-based university library with Django

This project is a Django-based library web application developed as part of an independent study program at SoftUni Indonesia. This application is designed to support the management of a university library, with key features such as user data management, journal recording, and a role-based login system. Data is stored in a structured manner using PostgreSQL as the main database. For the interface, Bootstrap is used to ensure a responsive and user-friendly design.

Laravel 11, Filament, Service Repository Pattern: Shoe Web

This project is a shoe e-commerce web application developed using Laravel 11 and Filament as the admin panel, and utilizing MySQL as the database system. This application was created in order to follow a class at BuildWithAngga. The interface design is composed with Tailwind CSS to produce a modern, minimalist, and responsive appearance. The main features in this application include a product catalog, stock management, ordering system, and admin dashboard that facilitates the management of product data and transactions.
